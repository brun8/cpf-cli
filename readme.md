# cpf cli

no terminal é mais rapido

## instalação

se já tiver go instalado

```bash
go build
```

ou

```bash
go install
```

## uso

pra gerar um cpf
```bash
./cpf 
```

pra gerar vários 
```bash
./cpf -n 10
```

pra gerar um com pontuação 
```bash
./cpf -p
```

pra gerar vários com pontuação 
```bash
./cpf -p -n 10
```

pra gerar um cpf de uma região específica
```bash
./cpf -r 1
```
