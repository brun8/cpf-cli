# cpf cli

no terminal é mais rapido

## instalação

se já tiver go instalado

```bash
go install
```

pra gerar um cpf
```golang
./cpf 
```

pra gerar vários 
```golang
./cpf -n 10
```

pra gerar um com pontuação 
```golang
./cpf -p
```

pra gerar vários com pontuação 
```golang
./cpf -p -n 10
```
